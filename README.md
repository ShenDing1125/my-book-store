# 我的書城
### 項目介紹 :

#### 「我的書城」是一個為移動端所開發的網頁平台。


### 項目技術 :

- #### 前端 :

  - TypeScript
  - Vue 3
  - Pinia
  - Vite

- #### 後端 :

  - Koa
  - MySql
  - Sequelize
  - Redis
